No 1 About Project

This web app is designed for students to help manage their job applications of an internship or a routine job.

This web app is implemented with Firebase, the application status are saved in firebase real-time database, the application documents are saved in firebase storage. The sign in section supports creating the user account with email and password or authentication using Google, Facebook, and Twitter. Users can add an application to the app by creating a new record or updating an existing record by changing its status. The documents section includes the resume, cover letter, and others such as transcripts and recommendation letters.
