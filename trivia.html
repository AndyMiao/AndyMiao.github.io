No 1 About Project

This is a two player trivia game built with Vue.js

The game gathers trivia questions from the Open Trivia Database with 3 kinds of difficulties that worth different points. Players answer the question alternately by choosing a question that worth 100, 300 or 500 points. If one of the players select a wrong answer, then the opponent would have the opportunity to answer the question. A player whose score is 1000 points ahead than the other one will win the game.
