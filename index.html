Projects

BiBli@ROBAUTO
BiBli, the educational robot with ROBAUTO.INC. The robot is built for potential Autism rehabihation, to improve the social skills of kids with Autism. So far, I have helped built two generations of BiBli. The first generation is now a permanant part of Longmont Library. And we are making prototypes, trying to cover all the libraries in Boulder, Longmont, Lyon, making BiBli the first robot to be checked out at a library. BiBli can be remotely controlled by a web user interface, equipped with a customizable video, audio media center to allow customers to build their own media curriculum.



Remote Owi Robotic Arm Edge
A remote, voice activated robotic arm. The robotic arm is built to help grab things in room. It can be mounted to a mobile robot base to move around. Additionally, to make it convinient to control. A mobile application was designed to control it remotely via Bluetooth. It was built with Owi Robottic Arm Edge and Raspberry Pi. I wrote a control algorithm with python to receive the command from remote tablet and then convert the command into the code which can be executed by arm control module. Then I can remote control the robot arm on my phone. After adding voice input for the app, I can even use voice to control the robot arm.
